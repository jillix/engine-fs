### `setProject(ev, data)`
Caches the project value.

#### Params
- **Event** `ev`: The event object.
- **Object** `data`: The data object:
 - `project` (String): The project name.

### `readFile(ev, data)`
Reads a file from project.

#### Params
- **Event** `ev`: The event object.
- **Object** `data`: The data object:
 - `path` (String): The file path.

### `writeFile(ev, data)`
Reads a file from project.

#### Params
- **Event** `ev`: The event object.
- **Object** `data`: The data object:
 - `path` (String): The file path.
 - `data` (String): The file content.
